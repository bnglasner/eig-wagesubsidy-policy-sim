## Logo Usage Rules (Quick Reference)

1. Never change logo colors.
2. Never stretch, compress, or distort.
3. Never add drop shadows, outlines, or effects.
4. Maintain clear space: minimum equal to the height of the "E" on all sides.
5. Minimum size: 1 inch / 72px wide.
6. Never place on a busy photographic background without a solid clear zone.
7. Never use the icon alone (without wordmark) in external materials without approval.
8. Always use SVG or @2x PNG for digital; EPS or CMYK JPG for print.

See `INFRA/docs/eig-brand-guidelines.md` for the full logo rules.
