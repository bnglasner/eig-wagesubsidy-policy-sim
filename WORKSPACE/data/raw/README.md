# WORKSPACE/data/raw

Document all source acquisition steps here.

For each source include:
- URL or access instructions
- retrieval date
- file naming conventions
- licensing/restrictions

Before implementation, a human must classify each source tier (`PUBLIC`, `LICENSED`, `RESTRICTED`, `HIGHLY_SENSITIVE`).
If any classification or restriction is unclear, ask the human for clarification.

Use `INFRA/docs/DATA_SOURCES_OUTPUTS_CATALOG.md` as the authoritative catalog for data-source and output metadata.
Use `INFRA/docs/DATA_GOVERNANCE_POLICY.md` for default governance and publication rules.
